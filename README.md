# PONG

Pong clone in C++ using raylib game library, and python GUI using PyQT

from /pong run game from python main after installing dependencies with:

```sh
        python3 main.py
```

## Dependencies

```sh
        brew install raylib
```
```sh
        brew install PyQt6
```
